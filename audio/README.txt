Adhan audio files for Ramadan Timer
====================================

Place these MP3 files in this folder for adhan playback:

  • fajr-adhan.mp3   – Fajr adhan (plays at Sehri end time)
  • iftar-adhan.mp3  – Adhan (plays at Iftar time)

You can use any adhan recordings you have. If files are missing,
notifications will still work; only sound will be skipped.
